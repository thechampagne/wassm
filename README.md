# WASSM

**This project is an edited version of [wassm](https://github.com/tsoding/wassm) to make it work with fasm.**

Web framework for x86_64 fasm.

## Quick Start

```console
$ make
$ make test
$ ./src/webapp 8080
```

The application serves files `./static/index.html` and `./static/main.css`.
